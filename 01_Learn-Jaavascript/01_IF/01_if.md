## Gunakan banyak if
→ Jika ingin mengecek semua syarat dan menampilkan semua error.

## Gunakan else if
→ Jika hanya ingin menampilkan satu hasil / satu error.

## Gunakan return
→ Untuk menghentikan function lebih awal sehingga tidak perlu else yang panjang.
